3. gitbook 생성

4. generate된 파일 구조

├── README.md            # 서적 소개(필수)
├── SUMMARY.md           # 목차 참조(선택)

5. book.json 생성

- gitbook 내에서 사용할 변수나 플러그인 등 설정하는 json type file

변수선언 예제

{
    "variables": {
        "BASE_URL": "https://111coding.github.io",
        "RAW_URL": "https://raw.githubusercontent.com/111coding/111coding.github.io/master"
    }
}

변수사용 예제

// {{ book.RAW_URL }}
